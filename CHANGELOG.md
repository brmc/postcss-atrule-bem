## [3.1.0] - 2017-11-11
### Added
- support for shortcuts options
  - _example:_

  ```
  /* with shortcuts: true; as option */
  @b block {
    @e element {
      @m modifier {
      }
    }
  }

  /* generates */
  .block {}
  .block__element {}
  .block__element--modifier {}
  ```
- support for separators
  - default: `{ element: '_', modifier: '-' }`
  - _example:_

  ```
  /* with element set to `_` and modifier set to `-` */
  @b block {
    @e element {
      @m modifier {
      }
    }
  }

  /* generates */
  .block {}
  .block_element {}
  .block_element-modifier {}
  ```
